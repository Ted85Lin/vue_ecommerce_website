# Vue.js 電子產品電商網站

本專案敘述：<br>
使用 Vue.js 3.x 製作的電子產品電商網站，使用 Vite 建立與打包專案，使用者可以選擇要購買的商品及數量，針對選擇資訊儲存在 Pinia 內，作為後續計算價格的依據，並在使用者調整商品數量後進行價格計算，擁有電商前台購物車模擬結帳的功能。

選用技術及工具：<br>
Vue.js 3.x、Vite、Bootstrap、Pinia、Vue Router
